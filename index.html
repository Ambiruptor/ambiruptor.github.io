Ambiruptor

The Ambiguation Interruptor

Introduction

Welcome on the website of the Ambiruptor project. The disambiguation problem consist of finding the correct sense of an ambiguous word in a given sentence. Our target is to use learning models in order to be able to guess the correct meanings of words with a highest possible accuracy.

Installation

To install the Ambiruptor project you have to clone the following repository.

git clone git@github.com:Ambiruptor/Ambiruptor.git

Then just use the Makefile to download / compile the basics needed files.
